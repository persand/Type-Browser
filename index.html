What

Type Browser helps you get a good overview of how a font looks on the web.

Read more about this project.

Keyboard shortcuts

Use ↑ ↓ keys to switch fonts and ← → keys to switch displays.

Source code

All available on GitHub. Feel free to fork and contribute.

Download latest stable release.
